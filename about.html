旅程是這樣開始

大學時期就讀藝術創意產業學系，
對藝術與設計持續關注且熱愛，
後來自學 UI 設計，
因機緣巧合為國立成功大學電機系實驗室製作官網，
解決了實驗室長期遇到的招生困境。

讓我體認到設計不只是視覺上的美，而是可以實際解決問題並帶來價值，
設計的力量讓我心生嚮往，於是轉職成為 UI 設計師。

至今累積四年相關經驗，
持續精進 UI 設計軟硬實力，
UX 是目前自學的領域，
致力於成為具成熟產品思維的設計師。

UI 設計師 + 切版

碩睿資訊有限公司 - 全職
2022 年 3 月 - 現在

碩睿資訊致力於提供優質且完整的電子資源服務，為學術研究、醫療事業、政府機關以及各產業提供重要的資訊來源，同時提供建立系統與製作專案的客製化服務。
  • 與企劃、設計師、工程師合作完成專案 。
  • 與企劃一起進行跟客戶的需求訪談，了解核心需求，給予適當的設計建議。

  • 專案完成後與企劃一起進行設計與功能的測試，確保符合設計規範與功能需求。

活動參與

Soking 需求訪談工作坊

2024.4.13

我的工作中不乏與客戶或是利害關係人的需求訪談，讓我因而想要提升訪談的技巧與知識，於是參與了群獸之心的 「需求訪談工作坊：練習辨識需求的情境，定義有效問題框架」，學習利用正向表述進行需求訪談的方法，同理利害關係人的需求。

UX 三刀流使用者體驗設計課程

2024.3-2024.7（五個月）

我認為要能夠洞察需求、理解使用者才能發展出好的設計，因此「使用者體驗設計」是我持續關注與學習的一個領域。這是由國立台北商業大學任教的溫明輝教授開放的共學課程，專注於從「使用者」、「商業」、「數據」三個面向，發展與構思創新產品與服務設計的能力。
課程對我收穫非常大，讓我用俯視的角度理解 UX 在產品週期的不同階段扮演怎樣的角色，以系統性地理解 UX 的多面性，包括了宏觀的理論框架和知識架構。完成 18 週的課程，包含 期中報告 與期末考，最後取得結業證書。

AAPD 第 1 屆 UI 設計線上實戰營

2023.6-2023.8（三個月）

這是非常紮實且密集的課程，營期間超過 10 場直播教學、講座和活動，透過課程作業與實際專案輸出的高效學習機制。讓我在硬實力方面，包含 Figma 的操作、設計能力與美感大幅進步；軟實力方面，包含團隊協作、工程邏輯和產品思維也明顯提升。 最後完成每週個人作業與團隊專案，獲得結業證書。若想了解更多，歡迎前往觀看 我的課程心得 以及 個人作業 。

第三屆捷運盃黑客松

2023.3

這一屆的主題聚焦在「台北捷運 Go App」的 UIUX Redesign，雖然提案沒有入圍，但我從觀摩入圍者的報告中習得很多，如何透過設計思考與能力，嘗試解決真實世界中遇到的問題；從 App Store 的留言分析或是問卷調查理解使用者的需求與痛點；再定義出甚麼是值得被解決的問題後，發展各種解決方案。讓我更加深刻認知到 UI 不只是設計出自我滿足的好看介面，如何洞察使用者，並在商業世界中取得平衡尤為重要。
